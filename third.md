Step 1: Create Trigger
Add the trigger “On form submission.”
This trigger activates whenever a user submits the form.

Step 2: Add AI Agent
Click the + icon and add AI Agent1.
Connect the form submission trigger to the AI Agent input.

Step 3: Select Chat Model
In the AI Agent settings, choose Chat Model.
Select Google Gemini Chat Model1.
Connect the model to the AI Agent’s Chat Model port.

Step 4: Configure Memory and Tools (Optional)
Add Memory if conversation history or data storage is required.
Add Tools if the AI needs to perform extra actions like accessing databases or APIs.

Step 5: Add Gmail Action
Click + after the AI Agent.
Select Gmail → Send a message.
Enter:
To: Recipient email
Subject: Example – “Form Submission Notification”
Message: Email content generated by the AI Agent

Step 6: Save and Test
Save the workflow.
Submit the form once.
The process runs automatically:
Form submission → AI Agent processes data → Gmail sends automatic email.
